dyld: Library not loaded: /System/Library/Frameworks/ModelIO.framework/XXXX问题解决

May 5, 2016

29号用朋友4s（iOS 8.4.1）真机调试时，编译时没问题，运行时程序直接奔溃，控制台打印出现了这样一个问题

后来Google了一下，网上也有人遇到过，是因为导入的框架引起的，将ModelIO.framework这个框架的status改为可选的即可(Optional)
